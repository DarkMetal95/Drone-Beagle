# Liste des commandes
Toutes les commandes sont en minuscule

	't': Altitude +
	'g': Altitude -
	'z': Déplacement avant
	'q': Déplacement latéral gauche
	's': Déplacement arrière
	'd': Déplacement latéral droit
	'k': Déconnection bluetooth
	'l': Extinction de la BBB
	'a': Activation du mode débug
